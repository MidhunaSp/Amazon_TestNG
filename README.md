# 🛒 Amazon Automation - TestNG

This project contains automation scripts for testing Amazon website functionalities using **Selenium WebDriver** and **TestNG**.

---

## 📌 Features

- Product search automation
- Filtering and sorting validation
- Product details verification

---

## 🛠️ Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- Eclipse IDE

---

## 📂 Project Structure

src/
 ├── main/java
 ├── test/java
testng.xml
pom.xml

---

## ▶️ How to Run

1. Right-click on `testng.xml`
2. Select **Run As → TestNG Suite**
